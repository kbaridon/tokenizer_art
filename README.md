Why using Metamask ?
--> Metamask is the main wallet available on the market: it's easy, free to use, and everything is provided. For exemple, to use hardhat, I need to use my wallet, and Metamask provide a key for hardhat.


Why using Ethereum/AVAX ?
--> Ethereum is well-known for its stability, many peoples have worked on it, so ressources are easy to find. Using Ethereum also allow me to use AVAX: More specifically the Avalanche Fuji Testnet. AVAX is a developer-friendly token where we can easily get a testnet. I already know AVAX because I used it in others projects like ft_transcendance, so it's was an easy choice for me.


Why using Hardhat to do it automatically ?
--> Hardhat provides better debugging infos (which is great as a beginner), and is overall more beginner-friendly than Truffle... Moreover, I already used hardhat in Tokenizer.


Why using ERC721 ?
--> Because I decided to use Ethereum, I had to respect the standards of the block chain. For Ethereum, the standard for NFT is ER721.


Why using Solidity ?
--> One more time, I used Ethereum blokchain. And the main language used in Ethereum is... Solidity !
